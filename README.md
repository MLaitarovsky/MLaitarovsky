### About Me
 
Israeli Olympic swimmer turned software developer. I competed at the **Tokyo 2020 Olympic Games**, trained with the Israeli National Team, served in the IDF, and then earned my **B.S. in Computer Information Systems** from the University of South Carolina (2025).
 
I bring the same mindset to code that I brought to competition — obsessive preparation, performing under pressure, and relentless iteration. Currently building portfolio-grade full-stack applications while training for the **LA 2028 Olympic Games**.

#### [FlowForge](https://github.com/MLaitarovsky/flowforge) — Visual AI Agent Orchestration Platform ⭐ Capstone
Drag-and-drop platform for designing, executing, and monitoring multi-step AI pipelines in real-time. BYOK architecture, DAG execution engine, and pre-built workflow templates for legal analysis, research, and sales intelligence.
 
`Next.js 14` `TypeScript` `React Flow` `Zustand` `BullMQ` `Redis` `PostgreSQL` `Prisma` `ChromaDB` `Claude API` `OpenAI` `SSE` `Docker`
 
**Highlights:**
- Visual node-based canvas with 10 AI operation nodes (LLM, RAG, chunking, structured extraction)
- DAG execution engine with topological sort and parallel branch execution
- BYOK (Bring Your Own Key) with AES-256 encrypted API key storage
- Real-time execution monitoring via SSE — watch nodes light up as pipelines run
- Demo mode with pre-computed results for instant showcase
- 3 workflow templates: Legal Document Analyzer, Research Assistant, Sales Intelligence Brief
